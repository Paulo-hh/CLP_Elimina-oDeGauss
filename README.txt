Para rodar os códigos, deve-se:

  1) Fazer download do repositório
  2) Abrir o terminal de comando no local onde estão inseridos os programas
  3) Executar o comando de terminal referente à linguagem desejada (ver a seguir)
         -> atenção: <matrix_dimension> deve ser substituído pelo 
                     tamanho desejado para a matriz. 


Programa em C deve-se executar dois comandos:

  1) gcc gauss.c -o gauss            Compila e cria o executável  
  2) ./gauss <matrix_dimension>      Executa o programa


Programa em Rust:

  1) cargo run <matrix_dimension>


Programa em Golang:

  1) go run main.go <matrix_dimension>




Exemplo:
  go run main.go 100
